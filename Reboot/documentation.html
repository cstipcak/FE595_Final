Polarity Documentation

This function returns the Polarity score, between -1.0 to +1.0
This is accomplished by using the textblob polarity function
Function Execution: TextBlob(input_text).polarity

Subjectivity Documentation

This function returns the Subjectivity score, between 0.0 to +1.0, with 0.0 being objective and 1.0 being subjective
This is accomplished by using the textblob subjectivity function
Function Execution: TextBlob(input_text).subjectivity

PoS Documentation

This function returns the part of speech for each word provided
This is accomplished loop and the Textblob pos_tags function
Function Execution: TextBlob(input_text).pos_tags]

Noun Phrase Documentation

This function returns the noun phrases found in the string provided
This is accomplished by using the Textblob Noun_phrases function
Function Execution: TextBlob(input_text).noun_phrases

Spellcheck Documentation

This function attempts to correct the spelling based on a set of rules.
This is accomplished by using the textblob correct function
Function Execution: TextBlob(input_text).correct()

Language Detection Documentation

This function indicates what language the text is.
This is accomplished by using the textblob detect_languange function
Function Execution: TextBlob(input_text).detect_language()

Translation Documentation

This function performs translation to the language of choice, in this implementation it is French
If a translation is unavailable, the following message is displayed "No translation available" This is accomplished by using the textblob Translate function
 Link to TextBlob Translation and Language Detection
Function Execution: TextBlob(input_text).translate(to='fr')

Stemming Documentation

This function stems the words of all of the words entered
This is accomplished by using the textblob stem function
 Link to TextBlob stem
Function Execution: PorterStemmer.stem(input_text)

Definition Documentation

This function defines each word of the sentence entered
If a definition is unavailable, the following message is displayed "No translation available" This is accomplished by using the textblob defininition function
 Link to TextBlob Definition
Function Execution: (Word(x).definitions)
